Pozionista


Description


Search for all potion recipes in a mysterious valley, using an old map.

Use mushrooms, flowers, roots and herbs gathered in the woods to make magic potions.

Experiment with ingredients and discover new ways to make potions.

Sell the potions you create to potion stores and use that money to travel farther and farther.



Features

  • Use different ingredients to create the same potion.
  • Sell potions to stores.
  • The map is divided into areas, time passes when you move from one area to another and you cannot travel at night but have to sleep in an inn.
  • Use two other tools to create ingredients that you will use to make potions.
